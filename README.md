## Running locally

Clone the project:

```bash
git clone https://github.com/matheustrres/qrcode-gen-js.git
```

Enter the project directory:

```bash
cd qrcode-gen-js
```

Install the necessary dependencies:

```bash
pnpm install
```

Start the project:

```bash
pnpm start
```
